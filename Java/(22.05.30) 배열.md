# 배열

## 배열 생성과 선언

### 1. 배열 선언
- 문법 : ```타입[] 변수명``` 또는 ```타입 변수명[]```

### 2. 배열 초기화
- 생성된 배열에 처음으로 값 저장
```java
int[] score = new int[5]  // 크기 5인 int형 배열 생성
score[0] = 100;
score[1] = 90;
score[2] = 80;
score[3] = 70;
score[4] = 60;
```

### 3. 다차원 배열 생성
- 문법 : ```타입[][] 변수명``` 또는 ```타입 변수명[][]``` 또는 ```타입[] 변수명[]```

***

## 배열 활용

### 1. length 메소드
- 문법 : ```배열명.length```

### 2. 가변배열
- 다차원 배열에서 마지막 차수 크기 지정하지 않고 각각 다르게 지정
```java
int[][] score = new int[5][3]
```

### 2. 배열 복사
- for문 사용

- System.arraycopy() 사용
```java
System.arraycopy(arr1, 0, arr2, 0, arr1.length);
// arr1[0]에서 arr2[0]으로 arr1.length개의 데이터 복사
```
