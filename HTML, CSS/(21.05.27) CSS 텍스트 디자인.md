## 색상

### 속성명

- color : 색깔 변경

- background-color : 배경 색상

- border-color : 테두리 색상

### 속성값
- HEX(16진수로 RGB; Red Green Blue가 표현된 값) 또는 주요 색상의 이름

```css
.box {
  color: #155724; /* 글자 색상 */
  background-color: #d4edda; /* 배경 색상 */
  border-color: #c3e6cb; /* 테두리 색상 */
}
```

## 글꼴

### 속성명
- font-family : 글꼴(폰트) 설정

### 속성값
- ```"폰트명"```
- fallback : 표현하고 싶은 글꼴이 없을 경우 대체할 폰트를 쉼표 뒤에 입력 (입력 순서대로 fallback 적용)
- 웹 폰트 기술 : https://d2.naver.com/helloworld/4969726
- 구글 폰트 서비스 (link 태그로 쉽게 embed) : https://fonts.google.com/

```css
.emphasize {
  font-family: "SF Pro KR", "MalgunGothic", "Verdana";
  /* 입력 순서대로 fallback 적용 */
}
```

## 글자 크기

### 속성명과 속성괎
- 속성명 : ```font-size```
- 속성값 : 글자 크키
```css
.title {
  font-size: 24px;
}
```
