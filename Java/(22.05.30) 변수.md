# 변수

***

## 변수의 타입
- 기본형에서 boolean을 7개의 타입은 서로 형변환(Casting) 가능

### 1. 기본형 (Primitive Type) 8개
- 논리형 : boolean (true / false)

- 문자형 : char (변수당 하나의 문자만 저장)

- 정수형 : int, long, byte (이진데이터 다루기), short (c언어 호환용)

- 실수형 : 실수 값을 저장 - float, double

### 2. 참조형 (Reference Type)
- 기본형을 제외한 나머지 (String, System 등)

- 객체 주소를 저장

***

## 변수 선언방법
- 문법 : ```[타입] [변수명]```

```java
int score;
score = 100;

int score = 100;

String str = new String("abc");
       str = null;
```

## 변수 명명

### 1. 명명 규칙 (Namimg Convention)
- 대소문자가 구분되며 길이 제한 없다.

- 예약어(Reserved word)를 사용해서는 안 된다. (true는 불가능, True는 가능)

- 숫자로 시작해서는 안 된다.

- 특수문자는 '_'와 '$'만 허용한다.

### 2. 명명 규칙 권장사항
- Class 이름의 첫 글자는 항상 대문자 (변수와 메소드 이름의 첫 글자는 항상 소문자)

- 여러 단어 이름은 단어의 첫 글자를 대문자로 하기

- 상수의 이름은 대문자로 한다. 단어는 '_'로 구분한다.

## 변수, 상수, 리터널

### 1. 변수(Variable) : 하나의 값을 저장하기 위한 공간
- 문법 : ```타입 변수명```

### 2. 상수(constant) : 한 번만 값을 저장할 수 있는 공간
- 문법 : ```final 타입 변수명```

### 3. 리터럴(literal) : 그 자체로 값을 의미하는 것
