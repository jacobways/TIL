# 도커 컨테이너 다루기

## 환경변수

- 환경변수 입력하기
```t
docker run -e MY_HOST=jacob.com ubuntu:focal
```

- 파일에 환경변수 입력하고 불러오기
```t
docker run --env-file [파일명] ubutu:focal
```


## 실행중인 컨테이너에 명령어 실행
```t
docker exec [container] [command]
```

## 도커 네트워크 구조
- eth0 : 호스트에서 사용하는 기본 네트워크

- veth : 가상(virtual) eth

- docker0 : 도커 엔진(데몬)에 의해 기본 생성되는 브릿지 네트워크로, veth와 eth 간 다리 역할
