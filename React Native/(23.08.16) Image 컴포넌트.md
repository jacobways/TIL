# Image 컴포넌트

### 1. 기능 및 특징

- 이미지를 불러오는 컴포넌트

- HTML의 `<img>`와 유사함

### 2. 사용

- `source` props로 이미지 파일 불러오기 : `require(상대경로)` 입력

- `style` props로 이미지 파일 스타일 입력

```js
<View>
  <Image source={require("../images/test.png")} style={styles.image} />
</View>;

const styles = StyleSheet.create({
  image: {
    width: 100,
    height: 100,
    margin: 20,
  },
});
```
