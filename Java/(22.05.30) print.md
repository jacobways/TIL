# print

***

## 화면에 글자 출력

### 1. print
- 문법 : ```System.out.print()```

- 괄호 안의 내용을 출력하고 줄바꿈 하지 않음

### 2. println
- 문법 : ```System.out.println()```

- 괄호 안의 내용을 출력하고 줄바꿈 함

***

## printf
- 출력형식 지정가능

### 1. 숫자 지시자
- %d : 불리언 형식으로 출력
```java
system.out.printf("age:%d year:%d", 14, 2017)
// age:1 year:2017
```

- %o : 10진(decimal) 정수 형식 출력

- %x, %X : 16진(hexa - decimal) 정수 형식 출력

- %f : 부동 소수점 (floating-point)의 형식으로 출력

- %e, %E : 지수(exponent) 표현식 형식 출력

### 2. 기타 지시자

- %b : boolean 형식으로 출력 

- %c : 문자(character)로 출력

- %s : 문자(string)로 출력

- %n : 줄바꿈
```java
system.out.printf("age:%d year:%d%n", 14, 2017)
// age:1 year:2017\n
```
