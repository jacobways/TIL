# String 클래스

---

## String 클래스

- String 클래스는 char배열에 기능(메서드)을 추가한 것

---

## String 클래스 주요 메서드

### 1. 문자열의 위치(index) 반환

- 문법 : `String변수.charAt(index)`

### 2. 문자열 길이 반환

- 문법 : `String변수.length()`

### 3. 해당 범위의 문자열 반환

- 문법 : `String변수.substring(startIndex, endIndex)` : endIndex 값은 포함하지 않음

### 4. 문자열 내용 같은지 확인

- 문법 : `String변수.equals(문자열)`

### 5. 문자열을 문자배열(char[])로 반환

- 문법 : `String변수.toCharArray()`
