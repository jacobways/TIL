# box-shadow, text-shadow

---

## box-shadow

- 기능 : 블록 요소에 그림자 효과 적용

- 속성명 : box-shadow

- 속성값 : 가로축 세로축 퍼짐정도 색상

```css
.box1 {
  width: 300px;
  height: 300px;
  background: blue;
  margin: 100px;
  float: left;
  box-shadow: 10px 10px 30px black; /* 적용 */
}
```

---

## text-shadow

- 기능 : 텍스트에 그림자 효과 적용

- 속성명 : text-shadow

- 속성값 : 가로축 세로축 퍼짐정도 색상

```css
p {
  font-weight: bold;
  font-size: 100px;
  font-family: "arial";
  color: #111;
  margin: 100px;
  text-shadow: 30px 30px 10px #aaa; /* 적용 */
}
```
