# CSS 색상 (color)

***

## color 속성명 방식
- 참고 링크 : https://www.w3schools.com/colors/default.asp

### 1. color name
```css
{
  color: red;
}
```

### 2. hex (hexa : 16진수 방식)
```css
{
  color: #FF0000;
}
```

### 3. rgb 

```css
{
  color: rgb(128,128,128);
}
```

***

## 색상 (color)

### 1. 속성명

- color : 색깔 변경

- background-color : 배경 색상

- border-color : 테두리 색상

### 2. 속성값
- HEX(16진수로 RGB; Red Green Blue가 표현된 값) 또는 주요 색상의 이름

```css
.box {
  color: #155724; /* 글자 색상 */
  background-color: #d4edda; /* 배경 색상 */
  border-color: #c3e6cb; /* 테두리 색상 */
}
```
